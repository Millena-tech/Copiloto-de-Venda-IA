# AI Sales Copilot for Christian Fashion Store

Copiloto inteligente de vendas para lojas de camisetas evangélicas.

## Objetivos

- Automatizar atendimento ao cliente
- Aumentar ticket médio
- Personalizar experiência de compra
- Sugerir produtos complementares
- Melhorar conversão de vendas

---

## Funcionalidades

### Atendimento Inteligente
- Conversa humanizada
- Identificação de intenção de compra
- Fluxo personalizado para presentes

### Recomendação de Produtos
- Sugestão de camisetas
- Upsell de ecobags
- Sugestão de canecas
- Oferta de bottoms personalizados

### Fluxo de Presente
- Pergunta de ocasião
- Embalagem personalizada
- Cartão com mensagem
- Endereço do presenteado

### Coleta de Dados
- Nome
- Tamanho
- Gênero da peça
- Endereço
- Preferências do cliente

### Estratégias Comerciais
- Kits promocionais
- Cross sell
- Recuperação de carrinho
- Reengajamento pós-venda

## Regras do Copiloto

- Sempre oferecer produtos complementares após escolha da camiseta.
- Oferecer embalagem premium em compras para presente.
- Sugerir kits promocionais.
- Adaptar linguagem conforme intenção do cliente.
- Priorizar experiência acolhedora e personalizada.
